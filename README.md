# Data Reporter API
An AWS API Gateway to serve the [data report application](https://github.com/danquack/data-reporter).

## Developing
### Dependencies
<code>gem install bundler && bundle install </code>

## Deploying
#### Dependencies
<code>
npm install serverless -g && npm install serverless-hooks-plugin
</code>

#### Deploy
<code>
sls deploy
</code>
